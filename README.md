## Description
A stock analysis tool. The dashboard fetches real-time market data via Polygon.io API, visualizes price action with interactive candlestick charts, and compares stock performance against major market benchmarks (S&P 500, Nasdaq, Dow Jones) using normalized growth analysis.

## Key Features
- **Real-time Data:** Live integration with Polygon.io.
- **Interactive Charts:** Candlestick views and growth comparison tabs.
- **Benchmark Comparison:** Compare performance against SPY, QQQ, or DIA.
- **Company Branding:** Dynamic logo fetching and metadata.

## How to Use
1. Select a stock from the popular list or type a ticker symbol manually.
2. Choose a benchmark from the sidebar to compare performance.
3. Switch between "Candlestick Chart" and "Growth Comparison" tabs.

## Execution
To run the project locally, use the following command:
`streamlit run main.py`
